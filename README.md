## Description
A simple GUI utility to act as a local (offline) image search engine.
Creates an index/db containing [imagePath, imageTextContent] pairs for easy searching.

## DB Struct

| cc | Row0 |  Row1 |
|----------|----------|-----------|
|  cc  |   imagePath  |   imageText(keywords)   |
